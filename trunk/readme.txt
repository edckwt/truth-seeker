== Description ==

Truth Seeker aspires to be a unique and reliable refuge regarding the truth of the creation, the Creator Allah and His existence, and the purpose of life.

Features:

* Get the content (articles, videos, audios) of [Truth Seeker](https://truth-seeker.info/).
* Using Json API
* Using Cron Job.


PS: All the websites owned by EDC [(E-Dawah Committee)](https://edc.org.kw), For Support click to [Islam.com.kw](https://islam.com.kw/).


== Installation ==

* Upload the Plugin to your blog, Activate it.
* Then the content will be stored automatically in your database.
* The plugin will work automatically using Cron Job to get the recent updates daily or according to the time you set.
